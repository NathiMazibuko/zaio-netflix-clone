Streaming Service Clone (Frontend)A responsive, frontend-only clone of a modern streaming service landing page (inspired by Netflix). It is the second assignment of the Full Stack AI Engineer Bootcamp at Zaio Institute of Technology. This project demonstrates proficiency in building complex UI layouts, managing CSS styling, and implementing interactive JavaScript features like media carousels.

🚀 Features
Responsive Design: Fully responsive layout that adapts to desktop, tablet, and mobile screens.
Hero Section: An engaging banner section with a clear call-to-action.
Media Carousel: A functional content slider built with JavaScript to browse through different video/show thumbnails.
"More Reasons to Join" Section: Styled informational cards utilizing custom SVG iconography.
Clean UI/UX: Adheres to modern design principles, utilizing the Karla font family for clean typography.

🛠️ Technologies Used
HTML5: Semantic markup and structure.
CSS3: Custom styling, flexbox/grid layouts, hover states, and transitions.
JavaScript (Vanilla): DOM manipulation and carousel logic.

📂 Project Structure
```text├── index.html # Main HTML document ├── style.css # Main stylesheet ├── script.js # JavaScript for carousel and interactivity └── README.md # Project documentation```

💻 Getting Started
To view this project locally, you don't need any special build tools or servers.
1. Clone this repository to your local machine:```bashgit clone https://github.com/your-username/your-repo-name.git```
2. Navigate to the project directory.
3. Double-click index.html to open it in your default web browser, or use an extension like VS Code's "Live Server" for an optimal development experience.
